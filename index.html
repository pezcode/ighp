iTunes Global Hotkeys Plugin

iTunes Global Hotkeys Plugin (ighp) for Windows enables the user to trigger different actions using predefined shortcuts even if the player is minimized or hidden.

This is a fork of the original ighp by Stefan Cosma which is located at ighp.berlios.de. For a list of changes see History below, forked versions begin at 0.0.4.

Install

The plugin is distributed as binary package (a zipped .dll + readmes) or as an installer which automatically copies the .dll to the appropiate place.

If you downloaded the .dll or compiled it from source you must manually copy it to the iTunes plugins directory (C:\Program Files\iTunes\Plug-ins) and then configure the hotkeys.

Download

You can download the installer here: ighp_0.1.0.exe
Or get the binary package here: ighp_0.1.0_bin.zip.

You can download the source code in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/pezcode/ighp

Configuration

To customize the hotkeys you must open the settings dialog in iTunes:
Go to View > Visualizer, select Global Hotkeys and click the Options item.

To set a hotkey:

  • select the action you want to set
  • focus the textbox next to 'Hotkey'
  • press the combination you need
  • click 'Set'

To delete a hotkey:

  • select the action you want to remove
  • click 'Clear'

History

0.1.0

  • Settings dialog accessible from iTunes Visualizer menu
  • About dialog
  • Options are saved inside iTunes
  • New actions: Fast Forward, Rewind, Quit
  • Merged the two DLLs into one
  • Removed support for hotkeys involving the Windows key
  • Removed actions: OpenSettingsFile, OpenSettingsDialog, ReloadHotkeys

0.0.4:

  • Support for standard multimedia keys: MMPlayPause, MMStop, MMPrev, MMNext, MMVolUp, MMVolDown, MMVolMute

0.0.3:

  • Two new actions: OpenSettingsFile and ReloadHotkeys
  • Ability to modify and reload hotkeys without closing iTunes
  • Configuration file stored in user folder
  • Installer kit

0.0.2:

  • This release adds more actions

0.0.1:

  • First alfa release
